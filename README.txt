Battleship - by Nathan Wang, Caspian Macky, and Tazkia Afra
CSMC105: Professor Kumar
-------------------------------------------
The game of Battleship is incredibly fun and easy to follow even if you have never played it before!
The rules and objective will be displayed in the game. Simply type 'no' when the game asks if you know the rules.
To start the game, run main.py. That's it!

Good luck!
